# 44krei Site Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 从零搭建一个以 `ianneo.xyz` 为美术蓝本、以 44krei / DH1Q 内容为主体的个人网站。

**Architecture:** 使用 Astro 生成静态站点，内容由本地数据文件驱动，页面由小型组件组装。视觉重点放在窄版式、轻边框、Hero 动态背景、列表 hover 和联系弹窗，以最小文件集快速做出接近参考站的体验。

**Tech Stack:** Astro, TypeScript, 原生 CSS, Vitest, Astro build
